Description

It's a fan game that I developed to dedicate to Nanayang, a streamer that I like

The concept was created using Nanayang's meme, Nataker, and it contains various stories related to Nanayang's interactions with NPCs and objects

The basic manipulation and gimmick are based on the original Helltaker moves, and the added obstacle gimmick is derived from the dwelling map of the favorite game, Spelunky 2
